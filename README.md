# YouTube Clone

A front-end clone of YouTube built using HTML and CSS. This project replicates the YouTube homepage layout, including the header, sidebar, video grid, and responsive design.

## Features

- Responsive video grid layout
- Sidebar navigation
- Header with search and voice search
- Video preview cards with channel info and view stats
- Hover effects and tooltips

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Roboto)
